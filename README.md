# Bangalore-Metro-Route-Ticketing-service
A full-stack web application for Bangalore Metro route management and ticket booking, built using React for the frontend and Node.js with Express and MongoDB for the backend, enabling route search, fare calculation, and seamless ticket booking

## 📌 Project Description
This is a full-stack web application for managing Bangalore Metro routes and ticket booking.  
It allows users to search routes, calculate fares, and book tickets easily.

---

## 🛠 Tech Stack
### Frontend:
- React
- HTML
- CSS

### Backend:
- Node.js
- Express.js

### Database:
- MongoDB

---

## ✨ Features
- 🔍 Search metro routes between stations
- 💰 Automatic fare calculation
- 🎟 Ticket booking system
- 📍 Station and route management
- 🌐 REST API integration
